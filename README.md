# Dunkelblaue Beispiel-Website

Kurzanleitung:

- Öffne `index.html` im Ordner `code` in deinem Browser oder starte einen Live-Server in VS Code.
- Die Seiten sind: `index.html`, `about.html`, `projects.html`, `contact.html`.
- Styles und Skripte sind in `assets/style.css` bzw. `assets/script.js`.

Kontaktformular ist clientseitig simuliert (kein Backend). Zum echten Versand musst du einen Server oder Drittservice hinzufügen.
